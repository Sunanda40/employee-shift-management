A simplified web-based Employee Shift Management System built using Java Full Stack (Spring Boot + JSP + MySQL).
This application helps small teams manage employee work shifts efficiently by allowing administrators to assign shifts and employees to view their schedules.
The Employee Shift Manager is a full-stack web application developed using Spring Boot and JSP.
It provides:
User registration and login functionality
Role-based system (Admin / Employee)
Shift creation and assignment
Shift viewing for employees
MySQL database integration
Clean and user-friendly interface
